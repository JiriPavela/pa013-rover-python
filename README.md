# PA013 Mars Rover Example in Python

The repository contains a simple Mars Rover example project for demonstrating unit and integration
tests in Python.

The `init` folder contains the initial version of the project corresponding to the state in which
we begin in the first lecture. The `final` folder then contains all the fixes and improvements
which we gradually introduced during the first and the second lecture.

## Installation

To install the project, navigate to the respective folder (either `init` or `final`) and run
`pip install -e .`. I recommend installing the project in a separate virtual environment so that
you avoid dependency collisions.
